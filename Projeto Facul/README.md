# Sistema de Gerenciamento de Obras

## Descrição do projeto

Este projeto consiste em uma aplicação web desenvolvida para auxiliar no gerenciamento de obras e construção civil. O sistema permite o controle de obras, materiais e informações básicas relacionadas ao andamento da construção, oferecendo uma interface simples, organizada e responsiva.

O objetivo do projeto é aplicar conceitos de desenvolvimento web utilizando HTML, CSS, JavaScript e Bootstrap.

---

# Tecnologias utilizadas

* HTML5
* CSS3
* JavaScript
* Bootstrap 5
* GitHub

---

# Instruções de execução

1. Faça o download dos arquivos do projeto ou clone o repositório:

```bash
git clone LINK_DO_REPOSITORIO
```

2. Abra a pasta do projeto.

3. Execute o arquivo `index.html` em qualquer navegador.

Não é necessário instalar dependências ou servidores adicionais.

---

# Organização das pastas

```bash
/projeto-obras
│
├── index.html
├── style.css
├── script.js
└── README.md
```

* `index.html`: estrutura principal do sistema;
* `style.css`: estilização da aplicação;
* `script.js`: funcionalidades JavaScript;
* `README.md`: documentação do projeto.

---

# Funcionalidades implementadas

* Página inicial do sistema;
* Menu de navegação responsivo;
* Cadastro de obras;
* Exibição de obras cadastradas;
* Controle visual de materiais;
* Barra de progresso das obras;
* Layout responsivo utilizando Bootstrap;
* Interface moderna e organizada.
